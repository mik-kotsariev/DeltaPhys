## Про проєкт
Цей репозиторій містить веб-обгортку (REST API) для модуля управління турами (ЛР 02–04), розроблену за допомогою Node.js та Express.

### Запуск локально:
1. Встановіть залежності: `npm install`
2. Запустіть сервер: `npm start`
3. Відкрийте браузер за адресою: [http://localhost:3000](http://localhost:3000)

### Основні Endpoints:
- `GET /health` — перевірка працездатності сервера
- `GET /api/config` — конфігурація та ліміти системи
- `GET /api/tours` — список створених турів
- `POST /api/tours/create` — створення нового туру
- `POST /api/sessions/register` — реєстрація учасника на тур
